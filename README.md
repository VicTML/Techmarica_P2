# Techmarica_P2
Este projeto é um banco de dados para controlar a produção de uma fábrica de equipamentos eletrônicos.
Foram criadas tabelas para funcionários, máquinas, produtos e ordens de produção.

O banco inclui:

Criação das tabelas (DDL)

Inserção de dados (DML)

Consultas com SELECT e JOIN

VIEW para consulta consolidada

Stored Procedure para cadastrar uma nova ordem automaticamente

Trigger para alterar o status para FINALIZADA quando a data de conclusão é preenchida
